# 日志系统说明

## 📋 概述

v1.3.2+ 版本引入了完整的日志管理系统，自动记录应用运行的所有关键信息。

## 📁 日志位置

### 开发环境
```
%USERPROFILE%\Documents\MouseControl\logs\
```

### 发布版本（便携版）
```
<应用目录>\logs\
```

## 📝 日志文件命名

- 格式：`app_YYYYMMDD.log`
- 示例：`app_20241121.log`
- 每天一个日志文件
- 自动清理7天前的旧日志

## 🔍 日志级别

| 级别 | 说明 | 使用场景 |
|------|------|----------|
| **DEBUG** | 调试信息 | 详细的点击统计、参数变化 |
| **INFO** | 一般信息 | 启动、停止、配置变更 |
| **WARNING** | 警告 | 非致命错误、权限问题 |
| **ERROR** | 错误 | 异常、崩溃 |

## 🛠️ 查看日志

### 方法1：使用辅助脚本（推荐）
```bash
# 双击运行
查看日志.bat
```

自动显示最新日志的最后50行。

### 方法2：手动查看
```bash
# 打开logs目录
cd logs

# 用记事本打开最新日志
notepad app_20241121.log
```

### 方法3：使用PowerShell
```powershell
# 查看最后50行
Get-Content logs\app_20241121.log -Tail 50 -Encoding UTF8

# 实时监控（类似tail -f）
Get-Content logs\app_20241121.log -Wait -Tail 10 -Encoding UTF8
```

## 🔧 日志配置

### 修改日志级别
编辑 `lib/logger_service.dart`:
```dart
_logger = Logger(
  printer: _CustomLogPrinter(),
  output: _FileOutput(_logFile!),
  level: Level.debug,  // 改为 Level.info 减少日志量
);
```

### 修改保留天数
编辑 `lib/logger_service.dart`:
```dart
await _cleanOldLogs(logDir, keepDays: 7);  // 改为更大的值保留更久
```

## 🐛 故障排查

### 问题1：找不到日志文件
**原因**：程序可能没有成功启动

**解决**：
1. 使用"调试启动.bat"查看错误
2. 检查是否有权限创建logs目录
3. 尝试以管理员身份运行

### 问题2：日志文件为空
**原因**：日志系统初始化失败

**检查**：
1. 磁盘空间是否充足
2. 杀毒软件是否拦截文件写入
3. 目录权限是否正确

### 问题3：日志太多磁盘占用大
**说明**：正常情况，每个日志文件约1-10MB

**解决**：
- 自动清理7天前的日志
- 手动删除旧日志文件
- 减少日志级别（改为INFO或WARNING）

## 💡 最佳实践

### 用户反馈问题时
1. 要求用户运行"查看日志.bat"
2. 复制日志内容
3. 或者直接发送日志文件

### 开发调试时
1. 使用DEBUG级别查看详细信息
2. 关键操作前后添加日志
3. 异常必须记录到ERROR级别

### 日志内容建议
```dart
// ✅ 好的日志
LoggerService.instance.info('开始自动点击 - 目标: ($x, $y), 间隔: ${interval}ms');

// ❌ 不好的日志  
LoggerService.instance.info('开始');
```
