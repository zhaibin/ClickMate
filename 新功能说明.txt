========================================
  新功能：捕获坐标快捷键
========================================

版本: v1.2.0
更新日期: 2024-11-21

========================================
  新增功能
========================================

✅ 双快捷键系统

现在有两个独立的快捷键：

1. Ctrl + Shift + S  - 开始/停止点击
2. Ctrl + Shift + C  - 捕获当前鼠标位置

========================================
  使用方法
========================================

【捕获位置快捷键】

1. 移动鼠标到目标位置
2. 按 Ctrl + Shift + C
3. 当前位置自动填入X、Y坐标框
4. 显示绿色提示消息

无需点击"捕获"按钮了！

【开始/停止快捷键】

1. 设置好参数后
2. 按 Ctrl + Shift + S
3. 开始或停止自动点击

========================================
  主界面显示
========================================

状态卡片中现在显示两个快捷键：

┌────────────────────────────────────┐
│ [▶开始/停止: Ctrl+Shift+S]        │
│ [📍捕获: Ctrl+Shift+C]             │
└────────────────────────────────────┘

绿色标签 = 开始/停止
蓝色标签 = 捕获位置

========================================
  设置快捷键
========================================

点击右上角⌨️图标打开设置对话框：

┌────────────────────────────────┐
│ [开始/停止] [捕获位置]  ← 选项卡│
│                                │
│ 选择按键: A B C D E F ...    │
│                                │
│ 预览: Ctrl + Shift + ?        │
│                                │
│ [取消] [保存]                  │
└────────────────────────────────┘

1. 点击顶部选项卡切换要设置的快捷键
2. 选择新的按键
3. 点击"保存"

========================================
  使用场景
========================================

场景1：快速捕获多个位置
1. 按 Ctrl+Shift+C 捕获第一个位置
2. 设置参数并点击
3. 按 Ctrl+Shift+C 捕获第二个位置
4. 继续使用

场景2：游戏中快速开关
1. 提前设置好位置
2. 游戏中按 Ctrl+Shift+S 开始
3. 按 Ctrl+Shift+S 停止
4. 无需切换窗口

场景3：连续捕获测试
1. 移动到位置1，按 Ctrl+Shift+C
2. 移动到位置2，按 Ctrl+Shift+C
3. 移动到位置3，按 Ctrl+Shift+C
4. 快速更新目标位置

========================================
  优点
========================================

✅ 更快速 - 无需点击按钮
✅ 更方便 - 全局快捷键
✅ 更高效 - 两手操作
✅ 更直观 - 主界面显示两个快捷键

========================================
  注意事项
========================================

1. 两个快捷键必须不同
2. 需要管理员权限才能正常工作
3. 使用 START.bat 以管理员身份启动
4. 如果快捷键冲突，可以在设置中更改

========================================
  推荐快捷键组合
========================================

默认组合：
- Ctrl+Shift+S (开始/停止)
- Ctrl+Shift+C (捕获)

备选组合1：
- Ctrl+Shift+K (开始/停止)
- Ctrl+Shift+L (捕获)

备选组合2：
- Ctrl+Shift+F9 (开始/停止)
- Ctrl+Shift+F10 (捕获)
