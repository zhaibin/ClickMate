# 鼠标自动控制器

Windows鼠标自动点击工具，支持全局快捷键和点击历史记录。

## ✨ 主要功能

- 🖱️ 自动鼠标点击（左键/右键/中键）
- 🎯 精确位置捕获
- ⌨️ 全局快捷键（Ctrl+Shift+按键）
- 📊 点击历史记录（最近10次）
- 🎲 随机间隔和位置偏移

## 🚀 快速开始

### 开发运行
```bash
run_debug.bat
```

## 📖 使用说明

1. **设置目标位置** - 手动输入坐标或点击"捕获"按钮
2. **配置参数** - 设置间隔、随机范围、偏移
3. **选择按钮** - 左键/右键/中键
4. **设置快捷键** - 点击右上角⌨️图标（默认Ctrl+Shift+S）
5. **开始/停止** - 点击按钮或按快捷键

### 点击历史

窗口底部显示最近10次点击记录：
- **序号** - 1-10
- **时间** - HH:MM:SS
- **位置** - (X, Y)
- **按键** - 左键(蓝)/右键(橙)/中键(紫)

## ⚠️ 注意事项

1. **DLL文件** - 确保`mouse_controller.dll`在应用目录
2. **管理员权限** - 某些情况下需要以管理员身份运行
3. **快捷键冲突** - 避免使用系统保留的快捷键组合
4. **合法使用** - 请勿在禁止脚本的游戏中使用

## 🐛 常见问题

**Q: 快捷键不工作？**
- 以管理员身份运行`run_debug.bat`查看日志
- 检查是否显示"热键系统初始化: 成功"
- 更换其他不常用的按键

**Q: 打包后无法运行？**
- 确保DLL文件在同一目录
- 查看打包发布指南

**Q: 界面显示不完整？**
- 窗口固定为520x680，不要手动调整
- 重启应用
